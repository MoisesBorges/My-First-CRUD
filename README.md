Crypto Wallet

This is the Crypto Wallet, an app that you can register and keep up with your cryptocoins.

Made with

ruby 2.5.0
Rails 5.2.4.2
bootstrap 4.1.1
sqlite3

How to install?

0. bundle
1. rails dev:setup
2. rails db:migrate
3. yarn add bootstrap
4. rails s

Or you can see it on [heroku](https://shielded-anchorage-32301.herokuapp.com/)
